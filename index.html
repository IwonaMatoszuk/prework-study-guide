HTML

  • HTML stands for Hyper Text MarkUp Language and, is one of the three cornerstone languages for web pages
  • The !DOCTYPE defines the document type
  • The head element contains information about the webpage
  • The title defines a title for the document
  • The body element represents the visible content shown to the user.
  • The section element defines a section in a document
  • The h1 to h6 defines HTML headings
  • The p defines a paragraph
  • The br inserts a single line break

CSS

  • CSS stands for Cascading Style Sheets and, it is a programming language that developers use to define how webpage is presented to the users.
  • CSS selectors are used to find/select the HTML element, which needs to be styled.
  • The * selects all elements
  • The .class selects all elements assigned to the given class attribute.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The img selector assign property values to any image element in the markup
  • The link element creates the connection to the external file
  • The href attribute specifies the location (URL) of the external file

Git

  • Git is a free version control system.
  • git config --global user.name "your name": sets your username for every repository
  • git config --global user.email "your email address": sets your commit email address
  • git clone [url]: creates a copy of an existing repository into a new directory
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git revert (insert SHA): gets back to required version of the project

JavaScript

  • JavaScript is a high-level dynamic programming language, that adds interactivity to the webpage.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The console.log is a method to log the value of variable to the web console
  • An 'if' statement is conditional statement; code will only execute if the statement meets the condition or proved true
  • Array is a single variable that holds multiple values, related in some way
  • Index is a unique number, which identifies location inside an array
  • A 'for' loop repeats until a specified condition is met
  • A 'for' loop contains 3 statements: starting point (e.g. var x = 0), condition (x < array.lenght) and final statement, that allows the array to iterate over each item (x++ is a code to increase the number by 1 everytime the code loops)
  • After 3 'for' loop statements, we put the code to be executed for each loop
  • To comment out we use /* and */, any comments between will be ignored
  • Function ia a set of instructions that tells the computer, how to perform a certain task
  • There are two requirements in JavaScript function: 1. Define the function 2. Call the function
